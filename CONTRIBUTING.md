# Contributing to Instill 🚀

First off, thank you for considering contributing to Instill! It's people like you that make Instill a great tool for everyone.

## 🌈 How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template in GitHub Issues.
- Provide a clear description and steps to reproduce.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Explain why this feature would be useful to the community.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`pnpm test`).
5. Make sure your code lints.

## 🛠️ Local Development Setup

```bash
# Clone the repository
git clone https://github.com/xblaster/instill.git
cd instill

# Install dependencies
pnpm install

# Build the project
pnpm build

# Run tests
pnpm test

# Try it out locally
node dist/index.js init
```

## 📜 Code of Conduct
Please be respectful and professional in all interactions within the project.
